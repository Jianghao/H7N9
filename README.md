# 中国地区H7N9 Web GIS 发布系统
</center>

## 介绍
从4月3日起发布H7N9的最新信息，包括：
* 病例的发生地点
* 病例的统计表格

该系统每日早上更新一次，现在已经更新的时间节点包括：
*  18:00, April 04
* 20:00, April 04
* 08:00, April 05 确诊病例14例，其中死亡5例；
* 10:00, April 06 确诊病例16例，其中死亡6例；

## 数据来源
数据主要来源于各大新闻网站，如新华网，新浪网，凤凰网以及官方微博信息。
